# Lens
> Capturing moments, freezing time.

## 🚀 Live Demo
https://lens-1777486675.vercel.app

## ✨ Features
- **Masonry Gallery:** A highly responsive grid layout designed to showcase high-resolution photography with optimal visual flow.
- **Light-box Viewer:** An immersive, full-screen image viewing experience featuring smooth swipe navigation and focus mode.
- **Modern-Minimalist Aesthetic:** Clean typography and generous whitespace that keeps the focus entirely on your portfolio.
- **Staggered Animations:** Engaging entrance transitions powered by Framer Motion to elevate the user experience.

## 🛠 Tech Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion

## 🏗 Architecture
The project follows a modular architecture using the Next.js App Router, with core UI logic encapsulated in the `components/` directory and page-specific routing defined in the `app/` folder. This structure ensures a clean separation of concerns, making it easy to manage gallery states and individual page content.
